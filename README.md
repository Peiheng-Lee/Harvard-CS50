# CS50x: Introduction to Computer Science

Welcome to my CS50 learning repository! This workspace contains my notes, source code, and problem-set solutions for Harvard University's CS50x. 

As a student with a strong focus on finance and quantitative analysis, I am taking this course to build a rigorous foundation in computer science. Understanding how computers operate at a lower level is crucial for writing highly optimized code and developing efficient algorithmic models.

## Core Focus: The C Language

Currently, this repository heavily emphasizes the **C programming language** (Weeks 1-5 of the course). Rather than starting with higher-level abstractions, I dove deep into C to understand the mechanics of memory, algorithmic efficiency, and low-level data management.


## Repository Structure (C Programming Phase)

* **Week 1: C Basics** - Syntax, variables, conditionals, loops, and custom functions.
* **Week 2: Arrays** - Compiling processes, command-line arguments, string manipulation, and cryptography basics (e.g., Caesar/Vigenère ciphers).
* **Week 3: Algorithms** - Asymptotic notation and implementing sorting/searching algorithms in C.
* **Week 4: Memory** - Hexadecimal, pointers, dynamic memory allocation, and file I/O (e.g., recovering deleted JPEGs from a memory card).
* **Week 5: Data Structures** - Building a fast spell-checker by implementing a hash table to load and search a massive dictionary in C.
